## üì¶ Installation

### Prerequisites
- Node.js (v16 or higher)
- MongoDB (local or Atlas)
- Git

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/Dhanunjay2704/spend-log-fullstack.git
cd spend-log-fullstack
```

2. **Backend Setup**
```bash
cd backend
npm install
```

3. **Environment Configuration**
Create `backend/.env`:
```env
PORT=5000
NODE_ENV=development
MONGO_URI=mongodb://localhost:27017/spend_log
JWT_SECRET=your_jwt_secret_key
JWT_EXPIRE=30d
BCRYPT_SALT_ROUNDS=12
```

4. **Frontend Setup**
```bash
cd frontend
npm install
```

5. **Environment Configuration**
Create `frontend/.env`:
```env
VITE_API_BASE_URL=http://localhost:5000/api
```

6. **Run the Application**
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm run dev
```

7. **Access the Application**
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

## üì± API Endpoints

### Authentication
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `GET /api/auth/me` - Get current user
- `PUT /api/auth/profile` - Update user profile

### Transactions
- `GET /api/transactions` - Get user transactions
- `POST /api/transactions` - Create transaction
- `PUT /api/transactions/:id` - Update transaction
- `DELETE /api/transactions/:id` - Delete transaction
- `GET /api/transactions/stats/overview` - Get statistics

### Budgets
- `GET /api/budgets` - Get user budgets
- `POST /api/budgets` - Set budget
- `DELETE /api/budgets/:id` - Delete budget

### Savings
- `GET /api/savings` - Get savings goal
- `POST /api/savings` - Set savings goal
- `PUT /api/savings` - Update savings goal
- `DELETE /api/savings` - Delete savings goal
